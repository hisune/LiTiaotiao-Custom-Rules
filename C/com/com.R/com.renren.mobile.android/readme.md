## 基础规则

快速复制:
```
{"popup_rules":
    [
        {"id":"青少年模式","action":"我知道了"},
        {"id":"iv_dialog_main_activity_bg","action":"iv_dialog_main_activity_close"},
        {"id":"radiance","action":"iv_dialog_live_room_gift_bag_close"}
    ]
}
```

- [{"id":"青少年模式","action":"我知道了"}](#id青少年模式action我知道了)
