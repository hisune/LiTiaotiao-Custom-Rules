## 基础规则

快速复制:
```
{"popup_rules":
    [
        {"id":"开启推送通知","action":"iv_close"},
        {"id":"tv_nps_title","action":"img_nps_close"}
    ]
}
```

- [{"id":"开启推送通知","action":"iv_close"}](#id开启推送通知actioniv_close)
- [{"id":"tv_nps_title","action":"img_nps_close"}](#idtv_nps_titleactionimg_nps_close)
